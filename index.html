Acerca de la Aplicación

Esta aplicación utiliza múltiples algoritmos de aprendizaje automático para realizar pronósticos de números del Baloto basados en datos históricos.

Los modelos implementados incluyen XGBoost, LightGBM, redes neuronales y modelos híbridos avanzados que combinan las fortalezas de distintos algoritmos:

  • Híbrido LightGBM + Red Neuronal: Combina la eficiencia de LightGBM con la capacidad de aprendizaje profundo
  • Híbrido XGBoost + Red Neuronal: Une la robustez de XGBoost con las capacidades de abstracción de las redes neuronales
  • Híbrido XGBoost + LightGBM: Fusiona dos potentes algoritmos de gradient boosting con enfoques complementarios

Para obtener mejores resultados, se recomienda proporcionar datos históricos completos y ajustar los parámetros de cada modelo según sea necesario.
